# Player Manual – Marshal & Spy

## Getting Started

1. Open the game in your browser
2. **Register** a new account (or login if you already have one)
3. Choose a game mode:
   - **VS AI** — play against the computer
   - **ONLINE PVP** — create a game and wait for opponent
   - **HOT-SEAT** — 2 players on the same device

## Interface Guide

### Login Screen
- Enter callsign (username) and access code (password)
- Click "New recruit? Register" to create account
- Minimum: 2 chars username, 3 chars password

### Lobby
- **DEPLOY section** — start new game
- **ACTIVE OPS** — join existing online games
- **SAVED GAMES** — load or delete saved games
- **LEADERBOARD** — ELO rankings of all players
- **Profile** — click your username to expand settings (password change, email)
- **CHANGELOG** link at bottom

### Deployment Phase
1. **Unit list** on the right panel shows all 23 units to place
2. **Click a unit** in the list → valid placement hexes highlight green
3. **Click a hex** to place the unit → next unit auto-selects
4. **RANDOM DEPLOY** button — auto-places all units and confirms
5. **CONFIRM DEPLOYMENT** — appears when all units are placed
6. Units placed at higher levels get attack bonus (+1 for L1, +2 for L2)

### Battle Phase
1. **Select your unit** — click on it on the board or in the side panel
2. Available actions highlight on the map:
   - **Green hexes** — valid movement targets
   - **Red hexes** — attack targets
   - **Blue hexes** — special ability targets
3. **Click a highlighted hex** to perform the action
4. **SKIP TURN** — pass without acting
5. **ESC** or **CANCEL** — deselect current unit

### Unit Display
- **Your units**: blue circles with abbreviation and attack number
- **Enemy units (hidden)**: dark gray circles with "??"
- **Enemy units (revealed)**: red circles with type and attack shown
- **Red dot** on your unit = it has been revealed to the enemy
- **Pulsing ring** = last moved unit (cyan = yours, red = enemy)
- **Dashed trail** = path of last move

### Controls

#### Mouse (Desktop)
| Action | Control |
|--------|---------|
| Select/action | Left click |
| Pan map | Left drag |
| Zoom | Scroll wheel |
| Cancel selection | Escape |
| Reset view | Home |

#### Touch (Mobile)
| Action | Control |
|--------|---------|
| Select/action | Tap |
| Pan map | One finger drag |
| Zoom | Pinch with two fingers |

### Top Bar Buttons
- **⛶** — toggle fullscreen (auto-fits board to screen)
- **SAVE** — save current game with custom name
- **PANEL** — show/hide info panel (mobile only)
- **ABORT** — leave game (with confirmation dialog)

### Side Panel (Desktop)
- Shows your units during battle with attack values
- Shows casualties list
- Shows game event log at the bottom
- Hidden on mobile to maximize board space — toggle with PANEL button

## Saving & Loading
- **Autosave**: game saves automatically every turn
- **Manual save**: click SAVE in top bar, enter a name
- **Load**: from lobby → SAVED GAMES → click LOAD
- **Delete**: click DEL next to any save

## Rankings (ELO)
- Starting ELO: 1000
- Win against higher-rated player = bigger ELO gain
- Only **Online PVP** games affect rankings
- AI and hot-seat games don't change ELO
- Leaderboard visible in lobby

## Tips for New Players
1. Start with **VS AI** to learn the mechanics
2. Use **RANDOM DEPLOY** first, then customize placement as you learn
3. **Scout before attacking** — use Recon Drone to reveal enemies
4. **Protect your Assassin** — devastating on offense, vulnerable on defense
5. **Watch the enemy's last move** (pulsing ring + trail) to track their strategy
6. **Jammer is key** — prevents enemy from scouting your units
7. **Mines near citadel** slow down rushes
8. The **Engineer** can re-hide revealed units — use it to protect valuable units
